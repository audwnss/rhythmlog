# 🌿 RhythmLog (리듬로그)

> **"Forced productivity를 버리고 sustainable routine을 찾으세요."**
> 나의 조건에 맞춘, 지치지 않는 하루의 시작을 돕는 웹 서비스입니다.

**RhythmLog**는 무리한 계획과 미달성으로 인한 죄책감을 줄이고, 사용자의 현재 에너지와 감정 상태에 맞춘 '지속 가능한 리듬'을 제안합니다. 완벽함보다는 '가벼운 시작'에 초점을 맞춥니다.

## ✨ 주요 기능 (Core Features)

1. **오늘 컨디션 체크인 (Check-in)**
   - 오늘 하루의 에너지, 집중도, 감정 상태를 3번의 탭으로 가볍게 기록합니다.
2. **AI 기반 데일리 리듬 제안 (Routine)**
   - 체크인 데이터를 바탕으로 무리 없는 하루 루틴을 제안합니다. 
   - 에너지가 부족한 날에는 '미뤄도 되는 일(LATER)'을 명시적으로 분리하여 심리적 부담을 덜어줍니다.
3. **3줄 데일리 회고 (Reflection)**
   - 하루를 마감하며 좋았던 점, 아쉬웠던 점, 내일의 나에게 할 말을 짧고 부담 없이 3줄로 기록합니다.

## 📂 프로젝트 구조 (Directory Structure)

현재 프론트엔드 UI를 구성하는 기본 파일 구조입니다. 바닐라 HTML/CSS/JS 기반으로 작성되었습니다.

```text
rhythmlog/
├── css/                  # 글로벌 및 페이지별 스타일시트
├── images/               # 아이콘, 로고 등 이미지 에셋
├── pages/                # 핵심 User Flow를 담당하는 세부 페이지
│   ├── checkin.html      # [Step 1] 데일리 컨디션 체크인 페이지
│   ├── routine.html      # [Step 2] AI 기반 데일리 루틴 제안 페이지
│   └── reflection.html   # [Step 3] 3줄 데일리 회고 페이지
├── app.js                # 공통 인터랙션 및 상태 관리를 위한 메인 스크립트
└── index.html            # 리듬로그 메인 홈 (랜딩 페이지)
